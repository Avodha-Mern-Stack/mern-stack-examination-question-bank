# **PART C Questions**

## **Basic HTML Questions (1-10)**

### 1. Create a simple webpage with:
- A heading with your name
- A paragraph introducing yourself
- An ordered list of your 3 favorite hobbies
- An unordered list of your 3 favorite books/movies
- A table with columns: Subject, Marks, Grade

### 2. Design a student information form that includes:
- Name (text input)
- Email (email input)
- Date of Birth (date input)
- Course selection (dropdown: Computer Science, Mathematics, Physics, Chemistry)
- Year (radio buttons: 1st, 2nd, 3rd, 4th)
- Submit and Reset buttons

### 3. Create a timetable for a week with:
- Table headers: Day, 9-10 AM, 10-11 AM, 11-12 PM, 1-2 PM, 2-3 PM
- Fill in sample data for Monday to Friday
- Use rowspan/colspan for lunch break (12-1 PM) spanning all days

### 4. Build a simple recipe page with:
- Recipe title as main heading
- Image of the dish
- Ingredients as unordered list
- Cooking steps as ordered list
- Preparation time and servings information

### 5. Create a feedback/survey form with:
- Name and contact information
- Rating scale (1-5 stars using radio buttons)
- Comments (textarea)
- Checkboxes for "Would you recommend to a friend?" and "Subscribe to newsletter"
- Submit button

### 6. Design a product catalog page with:
- Company logo/name in header
- Navigation links: Home, Products, About, Contact
- At least 3 product descriptions (image, name, short description)
- Footer with copyright information

### 7. Create a personal portfolio page with:
- Header with your photo and name
- Navigation: About, Skills, Projects, Contact
- Skills section with progress bars (using divs)
- Projects section with at least 2 projects
- Contact form at the bottom

### 8. Build a simple blog post page with:
- Blog title and author information
- Featured image
- Article content with headings and paragraphs
- Related articles section (unordered list)
- Social sharing buttons

### 9. Design a login page with:
- Logo/company name at top
- Login form (username/email and password fields)
- Remember me checkbox
- Login button
- Links for "Forgot password?" and "Create new account"

### 10. Create an event registration form for a workshop that includes:
- Participant details (name, email, phone)
- Workshop selection (dropdown with 3 options)
- Date preference (multiple checkboxes)
- Special requirements (textarea)
- Payment method (credit card, PayPal, bank transfer - radio buttons)

## **Multi-Page Website Questions (21-30)**

### 21. Build a 2-page restaurant website:
- **Page 1 (Home):** Hero image, menu highlights, location, contact info
- **Page 2 (Menu):** Full menu categorized (Appetizers, Main Course, Desserts)
- Navigation that works between both pages
- Consistent styling across pages
- Footer with opening hours and social media links

### 22. Create a 3-page educational website:
- **Page 1 (Home):** Welcome message, featured courses
- **Page 2 (Courses):** List of courses with descriptions and durations
- **Page 3 (Contact):** Contact form and campus location map
- Use external CSS file for styling
- Responsive navigation menu

### 23. Design a 2-page e-commerce site:
- **Page 1 (Product Listing):** Grid of products (image, name, price, "Add to Cart")
- **Page 2 (Product Detail):** Detailed product view with description, specifications, reviews
- Shopping cart icon in header showing item count
- Consistent header/footer across pages

### 24. Build a 3-page personal blog:
- **Page 1 (Home):** Latest blog posts with excerpts
- **Page 2 (Blog Post):** Full blog post with comments section
- **Page 3 (About):** Author bio and social links
- Archive sidebar on blog pages
- Search functionality (static design)

### 25. Create a 2-page event management site:
- **Page 1 (Upcoming Events):** List of events with dates, venues, "Register" buttons
- **Page 2 (Event Details):** Detailed event information and registration form
- Countdown timer for next event
- Calendar view for event dates

### 26. Design a 3-page hotel booking website:
- **Page 1 (Home):** Hero section with booking search form
- **Page 2 (Rooms):** Room types with amenities and pricing
- **Page 3 (Contact):** Contact information and inquiry form
- Booking confirmation modal/popup
- Star rating system for rooms

### 27. Build a 2-page fitness/gym website:
- **Page 1 (Home):** Class schedule, trainer profiles, membership plans
- **Page 2 (Sign Up):** Membership registration form with plan selection
- Workout of the day section
- BMI calculator (static design)

### 28. Create a 3-page photography portfolio:
- **Page 1 (Gallery):** Grid of photo thumbnails (landscape, portrait, wedding categories)
- **Page 2 (Portfolio):** Full-screen image viewer with navigation
- **Page 3 (About/Contact):** Photographer bio and booking form
- Lightbox effect for images
- Image filtering by category

### 29. Design a 2-page real estate website:
- **Page 1 (Property Listings):** Property cards with images, price, location, features
- **Page 2 (Property Details):** Detailed property view with photo gallery, description, contact agent form
- Mortgage calculator (basic design)
- "Save Property" functionality (heart icon)

### 30. Build a 3-page news/magazine website:
- **Page 1 (Home):** Featured news, latest articles, categories
- **Page 2 (Article):** Full article with author info, publish date, related articles
- **Page 3 (Contact/About):** Editorial team and contact form
- Newsletter subscription form
- Social sharing buttons on articles
