# PART C Questions

## **Practical Implementation Questions**

## 1. Create a simple React component that displays "Hello, React Learner!" on the screen.

## 2. Create a React component that Pass your name and course name as props from parent to child component and display them.

## 3. Create two components – Header and Footer – and display them inside the App.js file.

## 4. Build a React application to manage a list of students.
**Requirements:**
a. Display a list of students with their name and roll number.
b. Include functionality to add a new student using component state.
c. Include functionality to delete a student from the list.

## 5. Write a React + Redux example that maintains a counter with Increment and Decrement buttons. Explain each part of the Redux setup (actions, reducer, store, and component).

## 6. Create a React form with the following fields: Name, Email, and Password. Implement form validation showing error messages for empty fields and invalid email format. On submit, display the entered data.

## 7. Implement a Todo application in React with the following features:
- Add new todos
- Mark todos as complete
- Delete todos
- Filter todos (All, Active, Completed)
- Show the count of remaining todos

## 8. Create a React application that fetches data from a public API (like JSONPlaceholder) and displays it in a table. Include loading and error states.

## 9. Build a React context for theme (light/dark mode) and implement a toggle button to switch between themes. The theme should be accessible from any component.

## 10. Create a React Router application with the following pages:
- Home page (route: "/")
- About page (route: "/about")
- Contact page (route: "/contact")
- User profile page with dynamic route (route: "/users/:id")
- 404 page for undefined routes

## 11. Implement a search filter component that filters a list of items based on user input. The list should update in real-time as the user types.

## 12. Create a React component that implements a pagination system for a large list of items. The component should show a limited number of items per page and navigation buttons.

## 13. Build a React custom hook called `useLocalStorage` that syncs state with localStorage. Demonstrate its use in a component.

## 14. Create a React application with drag-and-drop functionality to reorder a list of items.

## 15. Implement a real-time chat interface in React using WebSockets or Socket.io. The interface should allow users to send and receive messages.

## 16. Build a React component that displays a countdown timer with start, pause, and reset functionality.

## 17. Create a React application that implements infinite scrolling for a list of items, loading more items as the user scrolls to the bottom.

## 18. Implement a file upload component in React that allows users to select files, shows upload progress, and displays uploaded files.

## 19. Build a React dashboard with multiple widgets (charts, tables, statistics) using a grid layout system. Make the widgets responsive.

## 20. Create a React application with internationalization (i18n) support for at least two languages (English and Spanish). Include a language switcher.
